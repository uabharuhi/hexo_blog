Hexo 建立方法

資料來源
資料來源

裝Node js

用Node js的cmd 裝Hexo

初始化blog

還要安裝三個套件 (我不知道這三個套件幹嘛的)

在current directory 下面就會有ㄧ個 的資料夾
假設dirname = blog

去github開repository

就隨便取個名字,自己方便就好

在local 創建一個repository然後push

反正就常用的git 指令
記得要先git add不然不能順利push

調整github上面的 GitHub Pages

去setting
左邊的Options
GitHub Pages 那個 section
調整自己要deploy (推送)的branch
調完之後記得儲存= =

產生blog

自己在local看爽的
沒問題之後就可以deploy到剛剛github開的repository

deploy

在第一次deploy
要先裝這個

這裡有坑爹的地方
首先要先到blog(最上面那層的)
_config.yml
調有的沒的
裡面最重要的是
url:github site的url

以下要注意

root : 預設是/
但按照yml檔案上面的comment
應該要改成 /blog
反正就是自己repo的資料夾名稱

在deploy:那邊
type: git
repository: 就push用的url
我是用ssh
git@github.com:/.git
用 http 也可以 不過之後要狂打密碼 頗煩
branch: master <–這裡的branch要和github page那邊source設定的一樣
然後deploy

應該會成功 沒成功的話可能是我漏了什麼 應該是沒有
剩下就靠自己的靈性了

阿好煩 打完了
